## General info
QR Code Generator with Rails. 
	
## Technologies
Project is created with:
- Ruby 2.6.6
- Rails 6.0.3
- Webpack 4.0
- rqrcode
- CSS/SCSS
- Bootstrap
- Fontawesome
- Work with `.erb` files
	
## Setup
```bash
bundle install
```

To start (usually on port `3000`):

```bash
rails s
```
